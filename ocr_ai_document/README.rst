Odoo OCR Using AI - Invoices, Bills, Purchase & Sale Orders
===========================================================
This module integrates AI-powered OCR technology with Odoo to automatically
extract data from documents and create business records.

Users can upload PDF or image documents such as invoices, bills, purchase
orders, or sales orders. The OCR engine processes the document, extracts the
relevant information, and automatically generates the corresponding record in Odoo.

Installation
============
- www.odoo.com/documentation/19.0/setup/install.html
- Install the custom addon **ocr_ai_invoice**
- Configure the OCR API credentials in the settings
