API Configuration

Navigate to Settings → OCR AI Integration → API Configuration. Add your fynix.ai Authentication Token and set the Server URL to https://ai.fynix.app/tus_ocr_api.

Model Configuration

Navigate to Settings → OCR AI Integration → Model Configuration. Select the model (e.g. Journal Entry) and configure the dynamic field mapping to tell the AI which fields to extract.

Create via OCR Button on Vendor Bills

The "Import via OCR" button appears on the Vendor Bills list. Click it to open the upload wizard and start the OCR extraction process.

Upload Document Wizard

The Import via OCR wizard allows you to upload a PDF, PNG, JPG, or JPEG document (max 2 MB). Click "Send to OCR" to process the document with AI.

AI Response Preview

After sending to OCR, the Response Preview tab shows the extracted data in JSON format. You can also preview the uploaded file and check the token usage details.

Auto Created Vendor Bill

After OCR processing, a draft Vendor Bill is automatically created with all extracted fields pre-filled — vendor, invoice date, due date, reference, line items, quantities, prices and taxes.

Import via OCR on Purchase Orders

The OCR button is also available on the Purchase Orders list. Upload a purchase document and AI automatically creates a draft Purchase Order with all fields filled.

Import via OCR on Sale Orders

The OCR button also appears on the Sale Orders list. Upload a sales document and AI automatically generates a draft Sale Order with extracted customer and product details.

Frequently Asked Questions

What does this module do?

This module integrates AI-powered OCR into Odoo so you can upload any vendor invoice, customer invoice, purchase order, or sale order PDF/image. The AI automatically extracts all key fields and creates the corresponding draft record in Odoo — eliminating manual data entry completely.

How do I get the fynix.ai API token?

Register a free account at https://ai.fynix.app. After login, go to the OCR Dashboard and copy your Authentication Token. Purchase OCR tokens from https://ai.fynix.app/shop (2 Million tokens for $20). Then paste the token in Odoo under Settings → OCR AI Integration → API Configuration.

What file formats are supported?

The module supports PDF, PNG, JPG, and JPEG formats. The maximum file size is 2 MB per document. For best results, use clear, high-resolution scans or digital PDFs.

How does the field mapping work?

Go to Settings → OCR AI Integration → Model Configuration. Select the model (e.g. Journal Entry for invoices/bills), then add field mapping rows. Each row maps an "AI Field Title" (what the AI will extract) to an Odoo field. For relational fields like invoice lines, you also specify the child fields (e.g. label, price_unit, quantity). No coding is required.

Does it work on Odoo Community?

Yes! This module works on both Odoo 19.0 Community and Enterprise editions, as well as Odoo.sh. Unlike Odoo's built-in document digitization which is Enterprise-only, this module brings AI OCR to Community users as well.

How many tokens does each document scan use?

Token consumption depends on the document complexity. A typical invoice scan uses approximately 500–2000 tokens. With 2 Million tokens ($20), you can process approximately 1000–4000 invoices. You can track exact token usage per request in the Token Usage tab of the wizard.
