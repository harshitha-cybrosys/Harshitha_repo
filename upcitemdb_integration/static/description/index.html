What does this module do?

This module allows you to create new products in Odoo by simply entering or scanning a UPC barcode. It fetches product data from the UPCitemdb API and auto-fills all relevant fields in the product form.

Which barcode formats are supported?

The module supports standard UPC-A (12-digit) and EAN-13 (13-digit) barcodes, which are the most common formats found on consumer products worldwide.

What happens if the UPC is not found?

If the barcode is not found in the UPCitemdb database, a validation error "Invalid UPC" will be displayed. The trial API allows approximately 100 requests per day.

Does it work with a barcode scanner device?

Yes! You can use a physical barcode scanner to scan the UPC directly into the barcode field in the Odoo product form. The onchange will trigger automatically once the value is entered.
