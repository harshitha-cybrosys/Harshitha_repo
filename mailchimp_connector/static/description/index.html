Mailchimp Connector

Seamlessly synchronize your contact lists, campaigns, and templates between Odoo and Mailchimp. Import and export audiences, members, and email templates with a single click — or let Auto Sync handle it automatically.

Key Highlights

Import Lists / Audiences

Import Lists and Audiences from Mailchimp directly into Odoo Mailing Lists with a single click — including all contact and campaign defaults.

Import Member / Contacts

Import Member contacts from Mailchimp audiences into Odoo mailing contacts seamlessly, with address and country details preserved.

Import Templates

Import email templates from Mailchimp into Odoo and use them directly in your mass mailings via the template selector on the mailing form.

Import Campaigns

Import Campaigns from Mailchimp into Odoo UTM Campaigns for unified tracking and reporting across both platforms.

Export Lists / Audiences

Export mailing lists and their contacts from Odoo to Mailchimp, creating new audiences or updating existing ones intelligently.

Auto Sync

Enable Auto Sync on your Mailchimp account in Odoo and a scheduled action will automatically keep audiences, templates, and members in sync at regular intervals.

Python Package Requirement

Before installing this module, the following Python package must be installed on the Odoo server by the system administrator: 

pip install mailchimp-marketing

After installing the package, restart the Odoo server before installing this module. For Odoo.sh deployments, this is handled automatically via the included requirements.txt.

Get API Key from Mailchimp

Login to your Mailchimp account at https://login.mailchimp.com/ and navigate to Account & Billing → Extras → API Keys to generate your API key.

API Key Configuration

Navigate to the API Key configuration section in your Mailchimp account settings.

Create New API Key

Generate a new API key in Mailchimp. Copy it immediately as it will only be shown once.

Mailchimp Menu in Odoo

Click on the Mailchimp menu in Odoo Email Marketing to access all Mailchimp-related features and configure your account.

Mailchimp Accounts

Click on the Mailchimp Accounts sub-menu to manage your Mailchimp connections in Odoo.

Add the API Key

Add the API Key copied from your Mailchimp account to connect it with Odoo.

Click Connect to Authenticate

After entering the API Key, click the CONNECT button to authenticate and verify the connection with your Mailchimp account.

Operations Menu

Click on the Operations menu to open the Import/Export wizard for syncing records between Odoo and Mailchimp.

Import & Export Wizard

The wizard provides options for importing and exporting lists, templates, and campaigns between Odoo and Mailchimp.

Import List / Audience

Select the Mailchimp Account, tick Import List/Audiences, and click IMPORT to bring your audiences into Odoo.

Imported List / Audience in Odoo

The imported List/Audience from Mailchimp is now visible in the Mailchimp Mailing List view in Odoo.

Import Templates

Select the Import Template option in the wizard to pull your Mailchimp email templates into Odoo.

Imported Templates in Odoo

Imported Templates from Mailchimp are visible in Odoo under the Mailchimp → Templates menu.

Import Campaigns

Select the Import Campaigns option to bring your Mailchimp campaigns into Odoo UTM Campaigns.

Imported Campaigns in Odoo

Imported Campaigns from Mailchimp are visible in Odoo under Reporting → Campaigns.

Select Export List / Audience

Select the Export List/Audience option in the wizard to push Odoo mailing lists and their contacts to Mailchimp.

Exported List / Audience to Mailchimp

The mailing list and its contacts from Odoo are now successfully exported to Mailchimp as an audience.

Use Template in Mail Body

Select an imported Mailchimp template directly on the mailing form to automatically populate the mail body.

Enable Auto Sync

Enable Auto Sync on the Mailchimp Account record so that audiences, templates, and members are automatically synced on a daily schedule.

Frequently Asked Questions

What does this module do?

This module connects Odoo with your Mailchimp account, allowing you to import and export lists/audiences, contacts, templates, and campaigns between both platforms seamlessly. It also supports Auto Sync via a scheduled action.

How do I get the Mailchimp API Key?

Log in to your Mailchimp account at https://login.mailchimp.com/, go to Account & Billing → Extras → API Keys, and click Create A Key. Copy the key immediately as it is only shown once.

Is a paid Mailchimp plan required?

Import features work on all Mailchimp plans including the Free plan. For Export, the free plan supports one audience only. The API key used must belong to the account Owner for full access.

Do I need to install any Python package?

Yes. The mailchimp-marketing Python package must be installed on the Odoo server before using this module. Run pip install mailchimp-marketing and restart the server. On Odoo.sh this is handled automatically via the included requirements.txt.

What is Auto Sync?

When Auto Sync is enabled on a Mailchimp Account in Odoo, a scheduled action automatically imports audiences, templates, and campaigns from Mailchimp and exports Odoo mailing lists back — all on a daily schedule without any manual intervention.
