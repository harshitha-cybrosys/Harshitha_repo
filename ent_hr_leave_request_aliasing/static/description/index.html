Configure Email Alias

Go to Time Off → Configuration → Settings. Set the Leave Email Alias prefix (e.g. "LEAVE REQUEST") and the allowed sender domain (e.g. "@cybrosys.info").

Configure Incoming Mail Server

Create a new incoming mail server in Settings → Technical → Incoming Mail Servers. Set the Create Record option to Time Off (hr.leave) and configure IMAP credentials.

Send Leave Request Mail

Employee sends an email with the prefix and leave type in the subject (e.g. "LEAVE REQUEST - Sick Time Off") and two dates in DD/MM/YYYY format in the body.

Fetch Emails in Odoo

Click "Fetch Now" on the incoming mail server. Only emails matching the configured prefix are fetched — all other inbox emails are ignored.

Auto Generated Leave Request

The leave record is automatically created in Time Off → Management → All Time Off with the correct employee, leave type, and dates all populated from the email.

Frequently Asked Questions

How can I automate leave requests through emails?

Configure your incoming mail server in Odoo to point to your leave request email address. Set the prefix and domain in Time Off → Configuration → Settings. Employees send emails with the prefix in the subject — Odoo automatically creates the leave record with employee, leave type, and dates.

Can I set a custom alias for leave request emails?

Yes. You can set a custom aliasing domain specifically for leave requests. This allows you to define a separate email domain to handle leave requests, making it easier to manage and track incoming requests related to employee leave.

How does leave type get selected automatically?

The module reads the email subject and matches a leave type name configured in Odoo. For example "LEAVE REQUEST - Sick Time Off" selects the Sick Time Off leave type automatically. If no match is found it falls back to the first available leave type for the employee's company.

What is the required email format?

Subject must contain the configured prefix followed by the leave type name (e.g. "LEAVE REQUEST - Sick Time Off"). The body must contain two dates in DD/MM/YYYY format on separate lines — first is the start date and second is the end date. The sender email must match an employee's work email in Odoo.
