POS Feature Manager
=======================
This module allows administrators to restrict or enable specific Point of Sale
features for individual employees. Restrictions can be configured per employee
directly from their profile, giving fine-grained control over what each
cashier is allowed to do during a POS session.

Configuration
============
1. Go to **Employees → Employees → [Employee Name] → Point Of Sale** tab.
2. Check or uncheck the desired permission booleans.
3. Save the record.
4. In **Point of Sale → Configuration → Settings**, enable
   **Log in with Employees** and assign employees to Basic, Advanced,
   or Minimal rights tiers.
5. Open a POS session and switch to the configured employee to apply
   restrictions.
