Key Highlights

Real-Time Tracking

Track all fleet vehicles live on a map. See current speed, battery %, accuracy and online status updated every 2 minutes via cron or instantly via webhook.

Trip History

Fetch and store full trip history per vehicle including start/end time, distance, average speed, max speed and GPS route points.

Event Alerts

Receive and log Traccar events — overspeed, geofence entry/exit, alarms, ignition on/off — directly inside Odoo with color-coded severity.

Webhook + Cron Sync

Dual data pipeline: instant push via Traccar webhook and fallback cron polling every 2 minutes ensures positions are never missed.

OpenStreetMap / Google Maps

Choose between free OpenStreetMap (no API key needed) or Google Maps. Live Leaflet map widget embedded directly in the vehicle form.

Easy Configuration

Single configuration screen with Test Connection button. Per-vehicle toggle to activate Traccar tracking. Works with self-hosted Traccar (Docker supported).

Traccar Configuration

Fleet → Traccar → Traccar Configuration. Enter your Traccar server URL and credentials, then click Test Connection.

Vehicle Form – Traccar Details

Enable Traccar on any fleet vehicle. Enter the Device Identifier (IMEI/uniqueId) from Traccar. The vehicle instantly shows live coordinates and status.

Live Tracking Dashboard

Fleet → Traccar → Live Tracking. See all active vehicles in one list with real-time speed, battery, status and one-click map links.

Trip History

Fleet → Traccar → Trip History. View all trips with distance, duration, average and max speed per vehicle.

Tracking Events

Fleet → Traccar → Tracking Events. All overspeed, geofence and alarm events logged with color-coded severity for quick review.

GPS Positions Log

Fleet → Traccar → GPS Positions. Every GPS fix stored with latitude, longitude, speed, altitude, accuracy and battery level.

Frequently Asked Questions

What is Traccar and do I need to pay for it?

Traccar is a free, open-source GPS tracking platform. You self-host it on your own server (a simple Docker command is enough). There is no subscription fee. The Traccar Client mobile app (Android/iOS) is also free.

How does the driver's phone send its location?

The driver installs the free Traccar Client app on their Android or iOS phone. They enter your Traccar server URL and their unique device ID. The app then continuously sends GPS positions to Traccar, which forwards them to Odoo.

How often are positions updated in Odoo?

Two methods run in parallel. A webhook pushes positions instantly when Traccar receives them. A cron job also polls every 2 minutes as a fallback. You can change the cron interval in the Traccar Configuration settings.

Do I need a Google Maps API key?

No. The default map provider is OpenStreetMap which is completely free and requires no API key. Google Maps is available as an optional alternative if you already have an API key.

Can I track multiple vehicles at once?

Yes. Simply enable "Activate Traccar" on each fleet vehicle and assign its Device Identifier. The Live Tracking Dashboard shows all active vehicles simultaneously with their current status.
