Frequently Asked Questions

What is the Enterprise Open HRMS Employee History module?

This module helps track an employee's complete work history including department changes, job position changes, salary updates, contract type changes, and hourly cost changes — all accessible from a dedicated 'Employee History' tab on the employee form.

Where can I find the Employee History records?

Navigate to Employees → select any Employee → go to the Employee History tab. From there you can access Job/Department History, Salary History, Contract History, and Timesheet Cost history using the respective buttons.

Does this module work with Odoo 19 Community edition?

This module is designed for Odoo 19 Enterprise edition. It requires the hr_payroll and hr_timesheet modules which are available in the Enterprise edition.

Who can view the employee history records?

HR Managers can view the history of all employees. Regular employees can only view their own history records. Attempting to access another employee's records will show an access error.

Will history be recorded for changes made before installing this module?

No. History tracking begins from the moment the module is installed. Changes made prior to installation are not retroactively captured. Only new changes saved after installation will appear in the history records.
