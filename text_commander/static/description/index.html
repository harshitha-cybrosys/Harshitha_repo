Frequently Asked Questions

What does this module do?

This module allows users to open any model, a specific record of a model, or create a new record using a command palette — all without navigating through menus.

How do I activate the command palette?

Press Ctrl + K from anywhere in the Odoo backend, then type ! to switch to the Text Commander namespace.

What can I do with the command palette?

You can: Open any Odoo model list, Open a specific record by name using /, Filter records by field value using of or in, and Create new records of any model.

What command syntax should I use?

Examples:
! Open Sale Order — opens list view
! Open Sale Order/S00001 — opens a specific record
! Open Sale Order of Customer Azure — filters by field
! Create Sale Order — opens a new record form
